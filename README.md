📌 Overview
A real-time Weather Dashboard built with asynchronous JavaScript and RESTful APIs. It allows users to search for weather by city name, view live metrics, and enjoy a modern, responsive UI with advanced features like geolocation, forecast, and dark mode.

🚀 Features
🔄 Fetch real-time weather data using Fetch API + async/await

⚠️ Comprehensive error handling for failed requests & invalid inputs

🌍 Search weather by city name

📍 Geolocation support (auto-detect user’s location)

🌡️ Metrics: temperature, humidity, wind speed, feels-like temp

🌅 Sunrise & sunset times

📊 5-day forecast with highs/lows

🔁 Unit toggle (°C ↔ °F)

🎨 Modern UI with dark/light mode toggle

💾 LocalStorage to save last searched city

🖼️ Dynamic background images based on weather conditions

⏰ Local time clock for searched city

♿ Accessibility: keyboard navigation, screen reader support
